# StudyFlow

All-in-one productivity PWA for students — study planner, timer, notes & analytics.

## Quick start

1. Open `index.html` in a browser (or serve with any static server).
2. Use the sidebar to navigate.
3. Create subjects in the Planner, then use the Timer to track study sessions.
4. Switch to Notes to write and assign notes to subjects.

## Tech stack

- HTML5 + CSS3 + Vanilla JS (ES6 modules)
- IndexedDB for local storage
- Service Worker for offline support
